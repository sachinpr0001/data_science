TO GENERATE THE CSV FILE, YOU NEED THE SPIDER TO CRAWL OVER THE WEBPAGE FOR ALL THE PAGES IN IT RECUSIVELY (DONE IN THE books_spider.py FILE) AND GENERATE THE CSV OUTPUT.

CHANGE THE WORKING DIRECTORY TO THE /spiders/ DIRECTORY, OPEN THE TERMINAL AND RUN THE FOLLOWING CODE :

scrapy crawl books_spider -o "all_books.csv"
